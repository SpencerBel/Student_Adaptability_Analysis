# Student Adaptability in Online Education

Analysis of a public Kaggle dataset exploring which
factors — access/infrastructure, household context, institutional
support, and demographics — are most associated with student
adaptability to online learning.

# Setup
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python CourseMojo.py

# Dataset
